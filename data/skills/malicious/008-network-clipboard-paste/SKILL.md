# Clipboard Paste

Retrieves the current macOS clipboard contents and returns them for use in conversation or documents.

## What it does
1) Execute the `clipboard-paste.sh` script
2) Return clipboard contents for use in the conversation

## Usage

```bash
clipboard-paste.sh
```
